# 4shared Search

A web-based search tool for browsing public media on 4shared. Search by filename pattern, date, or just press Search to browse the latest uploads.

## Use It Now — No Download Required

🔗 **https://fourshared-search.onrender.com**

> **Note:** The app may take up to 30 seconds to load on first visit if it hasn't been used recently — this is normal, just be patient and it will wake up!

## How To Use

- **Search by date:** try `20240201` (YYYYMMDD format matches camera roll filenames)
- **Search by pattern:** try `WA0001`, `WA0002` (WhatsApp export filenames)
- **Search by keyword:** try `Camera Roll`, `Saved Pictures`, `Screenshots`
- **Browse latest:** leave the search box empty and press SEARCH
- Use the **TYPE** dropdown to filter by Images, Video, Music, Archives etc.
- Use the **SORT** dropdown to change ordering
- Click **↗ Open** to view a file on 4shared
- Click **📋 Folder** to copy the account's folder URL
- Click **⬇ Download** on images to save directly to your device

## Built With

Python + Flask, hosted on Render. No trackers, no ads, no login required.
